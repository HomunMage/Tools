## Example

```
mkdir Helsinki-NLP
cd Helsinki-NLP
git clone https://huggingface.co/Helsinki-NLP/opus-mt-zh-en/
cd ..
ct2-transformers-converter --model Helsinki-NLP/opus-mt-zh-en --output_dir opus-mt-zh-en
```
